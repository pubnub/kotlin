# connectTimeout

`var connectTimeout: `[`Int`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-int/index.html)

How long before the client gives up trying to connect with a subscribe call.

The value is in seconds.

Defaults to 5.
